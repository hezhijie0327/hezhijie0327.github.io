如何安装并配置 aria2？

在本篇文章中我们以 Ubuntu 20.04 LTS 平台为例，其他系统操作相似

  1. 安装相关软件依赖： apache2 aria2 curl 
                                
sudo apt update && sudo apt install apache2 aria2 curl -y
  2. 下载 “配置文件生成 脚本 ” 并修改相关配置 
                                
sudo mkdir /etc/aria2
cd /etc/aria2
curl https://source.zhijie.online/aria2.conf/main/aria2.sh > /etc/aria2/aria2.sh
curl https://source.zhijie.online/aria2.conf/main/release.sh > /etc/aria2/release.sh
nano /etc/aria2/release.sh
  3. 运行以下命令，若无报错则代表运行成功： 
                                
bash /etc/aria2/release.sh && bash /etc/aria2/aria2.sh
  4. 下载并解压 AriaNg
  5. 将所有文件存放至 “/var/www/html” 文件夹中
  6. 在浏览器中输入本机 IP 地址，即可访问
