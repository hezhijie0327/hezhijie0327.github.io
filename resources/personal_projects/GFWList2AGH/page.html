如何使用 AdGuard Home 抗 DNS 污染？

在本篇文章中将讲述如何使用 AdGuard Home 抗 DNS 污染。

  1. 在浏览器中打开并登录 AdGuard Home 的网页管理界面
  2. 依次点击上方的 “设置” - “DNS 设置” 按钮
  3. 将 “ gfwlist2adguardhome_blacklist_full.txt ” 中所有内容替换至 “上游 DNS 服务器” 输入框内并点击 “保存配置” 按钮
  4. 请尽情享受无 DNS 污染的网络吧！

* 1. 在 v0.103.3-b6e58630 / v0.104 后，AdGuard Home 已支持外链上游 DNS 配置文件（暂不支持云端配置文件） *
* 2. 如需使用外链上游 DNS 配置文件，可将配置文件中 “upstream_dns_file” 的值改为 “ gfwlist2adguardhome_blacklist_full.txt ” 的绝对路径（如：/root/AdGuardHome/gfwlist2adguardhome_blacklist_full.txt） *
* 3. 如需修改上游默认 DNS 服务器，可修改 “ release.sh ” 文件中 “domestic_dns” 及 “foreign_dns” 的值 *
