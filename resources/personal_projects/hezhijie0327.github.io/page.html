如何使用 GitHub Pages 搭建个人网站？

在本篇文章中将讲述如何使用 GitHub Pages 搭建个人网站。

  1. 在登录 GitHub 后，点击右上角 “➕” - “New repository” 按钮
  2. 在 “Repository name” 输入框中输入 “用户名.github.io” 后点击 “Create repository” 按钮
  3. 点击 “creating a new file”，根据图示创建名为 “index.html” 的文件
  4. 在浏览器中访问 “用户名.github.io” 即可
