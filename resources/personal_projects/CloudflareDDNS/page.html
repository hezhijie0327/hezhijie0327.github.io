如何同步当前 IP 地址至 Cloudflare？

在本篇文章中将讲述如何使用 hezhijie0327/CloudflareDDNS 来同步当前 IP 地址至 Cloudflare。

  1. 打开并登录 Cloudflare 仪表盘
  2. 依次点击右上角 “人像” 图标 - “我的个人资料” 按钮
  3. 依次点击上方的 “API 令牌” - “查看” 按钮
  4. 在 “密码” 输入框内输入账户密码并完成 “人机验证” 后，点击 “查看” 按钮
  5. 复制文本框内的所有内容
  6. 打开终端并依次输入以下命令 (Linux / macOS 环境) 
                                
mkdir ./CloudflareDDNS
curl https://source.zhijie.online/CloudflareDDNS/main/CloudflareDDNS.sh > ./CloudflareDDNS/CloudflareDDNS.sh
nano ./CloudflareDDNS/CloudflareDDNS.sh
