如何安装并配置 AdGuard Home？

在本篇文章中我们以 Microsoft Windows 10 平台为例，其他系统操作步骤相似

  1. 下载 AdGuard Home 软件包
    操作系统 架构
    Darwin amd64 点击下载
    Darwin arm64 点击下载
    Docker amd64 点击前往
    Docker arm64 点击前往
    FreeBSD 386 点击下载
    FreeBSD amd64 点击下载
    FreeBSD arm64 点击下载
    FreeBSD armv5 点击下载
    FreeBSD armv6 点击下载
    FreeBSD armv7 点击下载
    Linux 386 点击下载
    Linux amd64 点击下载
    Linux arm64 点击下载
    Linux armv5 点击下载
    Linux armv6 点击下载
    Linux armv7 点击下载
    Linux mips64 点击下载
    Linux mips64le 点击下载
    Linux mips 点击下载
    Linux mipsle 点击下载
    Linux ppc64le 点击下载
    Windows 386 点击下载
    Windows amd64 点击下载
  2. 解压至指定文件夹，本文以 “C:\Program Files” 为例
  3. 下载 “ AdGuardHome.yaml ” 配置文件并将其保存至当前目录内
  4. 以管理员身份运行 CMD，并输入以下命令： 
                                
cd C:\Program Files\AdGuardHome
AdGuardHome.exe -s install
  5. 若返回以上类似值，即可在浏览器中打开并登录 AdGuard Home 的 网页管理界面
