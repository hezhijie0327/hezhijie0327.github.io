2020年07月09日 - Ubuntu 做软路由系统教程

本篇文章将讲述如何在 Ubuntu 环境中配置软路由系统。

系统环境：Ubuntu Desktop 20.04 LTS
所需软件：AdGuard Home，UFW

  1. 进入 “设置” 关闭所有网卡的 “自动连接”
  2. 编辑 /etc/netplan 文件
  3. 编辑 /etc/ppp/options 文件
  4. 编辑 /etc/sysctl.conf 文件
  5. 使用 nmcli 创建 PPPOE 拨号规则
  6. 将 AdGuard Home 作为 DHCP 及 DNS 服务器（ 安装教程 ）
  7. 配置 UFW 防火墙
  8. 重启即可
